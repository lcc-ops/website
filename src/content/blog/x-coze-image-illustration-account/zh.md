一个 AI 图文账号("图解万物 / 万物图鉴" 风格)据报在四个月内通过 Coze 工作流把跨平台矩阵做到 20 万粉,变现拆成两层:平台官方创作者分成 + 1000+ 元打包工作流的付费课程。新闻里那个 4 个月 20 万粉是传得最多的部分;结构上有意思的是第二层。下面是数学、工作流结构、案例没说清的部分。

## 案例说的数字

| 数量 | 值 |
|---|---|
| 公开跨平台粉丝 | 20 万+(4 个月) |
| 起步品类 | 猫狗(情绪价值高、受众广) |
| 生产工具 | Coze 工作流(prompt + 图像生成 + 文案 + 格式) |
| 视觉风格 | 扁平、角色化插画,跨帖子一致 |
| 变现第一层 | 发布平台官方创作者分成 |
| 变现第二层 | 1000+ 元打包工作流的付费课程 |
| 矩阵策略 | 跨平台账号跑同一条工作流 |
| 公开月总营收 | 未披露 |

案例把工作流当资产,把粉丝当渠道。两层是有意拆开的:第一层受算法波动,第二层跟平台无关。

## 为什么资产是工作流,不是 prompt

工作流相对单条 prompt 模版有三个不同的属性:

1. **稳定的参数集。** 工作流把调用哪个模型、温度多少、重试预算多少都编码进去。prompt 模版要 operator 自己记住这些参数。工作流消除了记忆依赖。
2. **管道化的前端。** 输入主题、输出成帖的工作流是另一种产品形态,不是「写一条 prompt 再手动写文案、格式、检查」。这条管道把「创意工作」压成「主题输入」。
3. **失败兜底。** 带重试、回退、拒绝逻辑的工作流能吸收模型版本漂移。从老博客抄来的 prompt 在底层模型更新时悄无声息地坏掉;工作流扛过模型更新的 operator 留住收入,prompt 漂移的 operator 丢收入。

这也是为什么打包的 Coze 工作流能卖 1000+ 元。买家买的不是模型本身,而是 operator 长期积累的决策。模型是便宜的那部分。

## 成本结构与单位经济

```
Coze 平台费:                          免费档或 0–99 元/月
图像生成模型单帖成本:                   0.05–0.50 元(看模型)
单帖 operator 工时:                   5–15 分钟(输入主题 + 审核)
矩阵单日产帖量:                       10–30 条(跨账号)
月工具成本(模型 + Coze):              50–300 元
月 operator 工时:                     40–120 小时
```

20 万跨平台粉的矩阵,创作者分成大致 **5000–20000 元/月**,看平台构成和互动率。课程收入是脉冲型的:开课月可以卖 100–500 单 × 1000+ 元,单月 10–50 万元;然后随品类成熟回落到 10–30 单 / 月的维护量。

第一层养 operator 的时间,第二层养 operator 的生意。案例把两层混在一起报,把第一层夸大、把第二层缩小。

## 矩阵是分销,不是护城河

跨平台账号跑同一条工作流是一种注意力放大模式,不是护城河。这个模式成立是因为:

- 每个平台有自己的受众池;三个平台各 5 万粉 = 15 万触达。
- 创作者分成在不同平台单价不同;operator 把产能倾斜到每播放单价最高的那个。
- 工作流产出跨平台可替换;开第三个平台的边际成本接近零。

矩阵做不到的是防工作流复刻。买了 1000 元课程的学员现在有了同一条工作流。矩阵护城河只在品类还没被填满、operator 是第一个动的人时存在。

## 案例没说的事

四点:

1. **第一层的真实收入。** 案例给的是粉丝和播放,不是分成人民币数。两者弱相关——20 万粉 × 3% 互动率 × 图文帖的实际分成,显著低于粉丝数暗示的现金量。
2. **课程退款 / 纠纷率。** 国内创作者平台上打包的 AI 工作流课有可考的「学员复刻失败 → 退款 / 投诉」曲线。案例给了价格,没给纠纷率。
3. **品类饱和曲线。** 猫狗插画是 3–6 个月的窗口。案例没有 12 个月粉丝曲线。
4. **平台对纯 AI 图账号的政策风险。** 主要国内创作者平台近 12 个月普遍收紧纯 AI 图政策。案例没给打标、披露、平台特定合规姿态。

## 结论

如果你能在建受众之前先把工作流搭起来,这条模型就跑得通。两层互相支撑:第一层验证品类,第二层拿走可累积的价值。如果先建受众、后搭工作流,你跑的是内容生意,不是工作流生意,矩阵护城河在一个平台政策周期内就会崩。

买家视角:新闻里那个粉丝数是营销层,不是收入层。Coze 工作流是资产;课程是变现;粉丝是漏斗。复刻粉丝但不复刻工作流,产出的是另一种生意,半衰期短得多。

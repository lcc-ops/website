拒付不是退款。这是 $15–20 费 + 产品成本 + 你的时间 — 哪怕你赢了。预防比抗辩便宜。

## 拒付两种类型

### 1. 未授权（持卡人欺诈）

有人用偷的卡从你这买东西。真实持卡人发现后拒付。开 3DS 的话责任转移给发卡行。没开 3DS 你承担损失。

**开 3DS 胜诉率**：~85%（发卡行赔）
**没开 3DS 胜诉率**：~40%（你赔）

### 2. 商户争议（买家反悔）

- **未收到货**（INR）— 买家声称产品没到
- **严重货不符**（SNAD）— 产品和描述差太多
- **服务未提供** — 数字产品、服务

这些要证据，不是欺诈过滤器。**证据充分胜诉率：60–80%**。

## 防范清单

### 销售前

- [ ] **所有交易开 3D Secure**。Stripe 和 PayPal 都支持。转化率降 3–5%，拒付降 50%+。
- [ ] **用 AVS**（地址验证系统）。账单地址和收货地址不匹配就是欺诈信号。
- [ ] **强制 CVV** — 永远不豁免。多数欺诈没有 CVV。
- [ ] **速度检查** — 同一 IP/卡 24 小时内 3+ 单就标记。
- [ ] **屏蔽高风险国家**（如果本来就不发）。Stripe Radar 有列表。
- [ ] **不要**强制签收 — 签收要求买家必须在场，通常上班时间，反而增加拒付。

### 结账时

- [ ] **清晰的账单描述符** — 用可识别名字。"店铺名 12345" 而不是 "TXN*987XYZ"。
- [ ] **订单确认邮件** — 立刻发，含订单号、商品、总价、预计到达。
- [ ] **发货确认邮件** — 标签创建就发，附物流号。
- [ ] **实时跟踪** — 包裹移动就发更新。

### 销售后

- [ ] **主动沟通** — 物流延误时，**买家联系你之前**先发邮件。INR 拒付降 40%。
- [ ] **简单退款路径** — 让"申请退款"一目了然。能轻松退款的买家很少拒付。
- [ ] **24 小时内回复所有消息** — 沉默就是拒付燃料。
- [ ] **宽松退货政策** — 30 天、免费退货。成本低于拒付费。

## 拒付申诉要交什么

### "未收到货"

1. 物流号+承运商
2. 承运商送达状态+送达日期+地点
3. 签收（如有）
4. 买家下单 IP 地址
5. 邮件往来（你发过的"订单呢？"回复）

### "货不符"

1. 原始商品页（截图+URL）
2. 实发商品照片
3. 供应商/厂家描述
4. 买家签收确认
5. 商品相关的历史沟通

### "未授权"

1. 3DS 认证记录（CAVV / ECI 值）
2. AVS 匹配（账单 = 收货地址）
3. CVV 匹配
4. 买家订单历史（老客或首单）
5. 设备指纹+IP 地理位置匹配账单地址

## 不要做的事

- **不要**退款又接受拒付 — 你会被扣两次（退款 + 拒付费）。
- **不要**发到和账单地址不同的地址 — 几乎必输。
- **不要**用通用拒付文本 — 每个争议要具体证据。
- **不要**错过截止日期 — 争议开起来立刻设日历提醒。
- **不要**和客户吵架 — 保持专业沟通，法律威胁适得其反。

## 预防挡不住的时候

一些拒付不可避免。追踪比率：

- **< 0.5%** 交易：健康
- **0.5–1%**：临界；审视流程
- **1–1.5%**：高；会被卡组织标记
- **> 1.5%**：进了监控项目；账户会被冻结

比率攀升时审计：

1. 新上的商品（有没有误导买家的描述？）
2. 物流承运商（延误导致 INR？）
3. 最近客服投诉（未解决问题变成拒付？）

"Dropshipping 零成本启动"是谎话。下面是真实店铺的支出。

## 完整成本堆

$50 产品 AliExpress dropshipping 店铺：

| 成本 | 金额 | 占营收 % |
|---|---|---|
| **收入** | **$50.00** | **100%** |
| 产品成本 | $12.00 | 24% |
| 到客户物流 | $4.50 | 9% |
| Shopify Basic | $1.30（分摊） | 2.6% |
| Shopify Payments | $1.75（2.9% + $0.30） | 3.5% |
| 应用（5 × $20/月 分摊） | $3.30 | 6.6% |
| 邮件工具 | $0.50 | 1% |
| **广告前小计** | **$26.65** | **53.3%** |
| **毛利润** | **$23.35** | **46.7%** |
| 广告支出（Google + Meta） | $10.00 | 20% |
| **退货前净利率** | **$13.35** | **26.7%** |
| 退货（8% 订单 × $50 × 30% 成本基础） | $1.20 | 2.4% |
| 拒付（1% × $15 费） | $0.15 | 0.3% |
| 退款处理（Stripe 收的退款费） | $0.20 | 0.4% |
| 客服时间（分摊） | $0.50 | 1% |
| **真实净利率** | **$11.30** | **22.6%** |

## 侵蚀利润的隐形成本

### 1. 慢物流退款

美国客户期望 5–7 天到货。AliExpress 物流 15–25 天。约 5% 客户申请部分退款（"我的订单呢？"）或全额退款。

每个慢物流投诉平均赔 $10。100 单/月就是 $50 退款，这些问题你阻止不了。

### 2. "货不符"投诉

客户收到 $12 AliExpress 产品后，约 3% 会发起"货不符"投诉。PayPal 和 Stripe 通常站买家。每单成本：$20 产品退款 + $15 拒付费 + $5 时间 = **$40/单**。

### 3. 广告素材成本

成功店铺每月测试 5–10 个素材。每个视频素材外包 $50–300，自己做 4–8 小时。多数新手没预算这个。

### 4. KOL / UGC 成本

用 KOL 或付费 UGC，每个创作者 $100–500。不追踪等于 ROI 未验证。

### 5. 订阅疲劳

试过不用的应用还在扣费。每月审计。

### 6. 货币转换损失

Stripe 货币转换收 1%。PayPal 收 3–4%。收美元但付人民币供应商，默认损失 2–3%。

### 7. 库存波动

Dropshipper 控制不了库存。广告跑一半产品断货。断货品浪费的广告：总广告预算的 5–10%。

### 8. 客服时间

每单产生约 0.1 小时客服时间。按 $20/小时全负荷算，$2/单。新手都低估。

## "爆品"陷阱

大师卖"爆品"清单。典型"爆品"强销 30–60 天，然后被同行复制热度消退。第 4 个月 ROAS 从 4× 跌到 1.5×，产品死了。

现实场景：
- 第 1–2 月：$10k 营收，25% 净利率 = $2,500 利润
- 第 3 月：$8k 营收，20% 净利率 = $1,600 利润
- 第 4+ 月：$4k 营收，10% 净利率 = $400 利润
- 需要新产品

任何时候店铺都需要 3–5 个活跃爆品维持营收。

## 为什么大多数 Dropshipping 店失败

1. **没复购客户** — 纯冷流量，每单获客成本一样。
2. **单产品依赖** — 一个爆品衰退，营收崩盘。
3. **低估 CAC** — 新手没算广告 + 素材成本。
4. **没品牌** — 同质化产品只能拼价格。
5. **物流慢** — AliExpress dropshipping 输给亚马逊的物流。

## 2026 年怎么赢 Dropshipping

- **用美国/欧洲供应商** — CJ Dropshipping、Zendrop 或本地 3PL。贵 20%，5 天到货，丢单更少。
- **建细分品牌** — 选一个品类把它做透。杂货店拼价格，品牌店拼认同。
- **第一天就建邮件/短信列表** — 复购客户比首单客户价值高 3 倍。
- **打包销售** — $50 客单价而不是 $25。广告花费一样，收入翻倍。
- **每周测 5 个产品** — 大多数会失败，这就是游戏。
- **追踪净利不是营收** — 营收是虚荣，利润率是理智。

CBM 和体积重是决定物流费用的两个数字。算错要么多付，要么收到承运商的调整账单。

## 什么是 CBM？

CBM = **立方米**。是货物的体积：

```
CBM = 长 (m) × 宽 (m) × 高 (m)
```

例：一个箱子 60cm × 40cm × 30cm
```
0.6 × 0.4 × 0.3 = 0.072 CBM
```

10 个相同箱子：0.72 CBM。

## 什么是体积重？

体积重（也称"泡重"）按货物占的空间算，不按质量算。轻但体积大的东西如果只按实际重量收费就太便宜。

```
体积重 = (L × W × H cm) ÷ 除数
```

除数取决于承运商和运输方式：

| 方式 | 除数 |
|---|---|
| 空运 | 6000 |
| 快递（DHL、FedEx、UPS） | 5000 |
| 海运散货（LCL） | 1000 |
| USPS First-Class | 9000 |
| 亚马逊 FBA 入仓 | 5000 |

## 承运商按两者取大

每票货，承运商都算两个数：

- **实际重量**（秤读数）
- **体积重**（体积 ÷ 除数）

你付**较大**的那个。这就是 **计费重**。

## 算例

一个 50cm × 40cm × 30cm 重 8kg 的箱子，DHL 出货：

```
实际重量：        8 kg
体积重：          (50 × 40 × 30) ÷ 5000 = 60000 ÷ 5000 = 12 kg
计费重：          12 kg
```

你付 12kg 的钱，不是 8kg。哪怕秤上只读 8。

## 什么时候体积重起作用

经验法则：产品**体积大但轻**（枕头、泡沫、灯罩）时，体积重总是赢。**密度高体积小**（书、金属件）时，实际重量赢。

| 产品类型 | 体积重 or 实际？ |
|---|---|
| 服装（压缩） | 实际 |
| 书 | 实际 |
| 泡沫、枕头、毛绒 | 体积重 |
| 灯罩、篮子 | 体积重 |
| 电子（小） | 实际 |
| 家具配件 | 体积重 |
| 保健品、化妆品 | 实际 |
| 充气产品 | 永远是体积重 |

## 怎么降低体积重

公式改不了，但包装可以改：

1. **用能装下的最小箱子** — 每厘米都算钱。能用 40cm 箱就别用 50cm。
2. **软货压紧** — 服装真空袋能减 30–50% 体积重。
3. **拆成多箱** — 两个 5kg 箱通常比一个 10kg 形状奇怪的箱更便宜。
4. **能折叠就折叠** — 折叠大幅降低体积。

## 算例：包装优化

10 件衬衫装 50×40×30cm 箱，每件 200g：

```
原箱：
  尺寸：50 × 40 × 30 cm
  实际重：2 kg
  体积重：60000 ÷ 5000 = 12 kg
  计费重：12 kg → 运费 ~$66（DHL）

压缩 + 小箱 (40 × 30 × 20 cm)：
  实际重：2 kg
  体积重：24000 ÷ 5000 = 4.8 kg
  计费重：4.8 kg → 运费 ~$26

省 60%
```

## 海运 LCL 特殊性

海运拼箱（LCL）的除数是 **1000** — 意味着体积重更狠。1 CBM 高密度货（1000kg 实际）打平。低于这个密度就是为体积买单。

多数 LCL 承运商还有 **最低 1 CBM** 起收费，哪怕你只出 0.3 CBM。

## 空运 vs 快递

| | 空运 | 快递（DHL/FedEx） |
|---|---|---|
| 除数 | 6000 | 5000 |
| 单价/kg | $4–8 | $6–12 |
| 适合 | 100kg+ 大货 | < 100kg 急件 |
| 时效 | 5–10 天 | 3–7 天 |
| 跟踪 | 基础 | 实时 |

快递除数更小（5000 vs 6000），体积重更高 — 所以快递对泡货更贵。

## 速查：什么时候用什么方式

| 货物画像 | 最佳方式 |
|---|---|
| < 20kg，急件 | 快递 |
| 20–200kg，密度高 | 快递或空运 |
| 100–500kg，任意密度 | 空运 |
| 200kg+ / 2 CBM+，不急 | 海运 LCL |
| 整柜（28+ CBM） | 海运 FCL |

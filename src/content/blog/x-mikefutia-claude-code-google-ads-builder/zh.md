一条高互动 X 帖（619 赞、526 回复）走了一遍 Claude Code 搭建：一个 URL 进去，出来一个完整可投的 Google Search 广告系列——关键词、广告组、每条标题、否定词，按付费搜索代理会用的结构组织。下面拆这个搭建实际做了什么、每个广告系列的单位经济学、套壳生意坐在哪里。

## 搭建实际做了什么

| 阶段 | 输入 | 输出 | 时长 |
|---|---|---|---|
| 爬取 + 提取 | 一个 URL | offer、价值主张、信任点、定价 | 1–3 分钟 |
| 关键词主题 | 提取出的 offer | 按意图分 5–12 个广告组 | 2–4 分钟 |
| 标题 + 描述 | 广告组 | 每组 15–30 条标题 + 4 条描述 | 3–8 分钟 |
| 否定词 + 出价 | 广告组 | 否定关键词清单、出价建议 | 1–3 分钟 |
| Editor CSV | 以上全部 | Google Ads Editor 导入文件 | <1 分钟 |

搭建输出一个 Google Ads Editor CSV，运营方能直接上传。首稿工作做完了。剩下的工作是 QA、转化追踪接线、上线后调优。

## 每个广告系列的单位经济学

| 项 | 成本 | 备注 |
|---|---|---|
| Claude 订阅 | $20–200/月固定 | 分摊到每月 10–40 个广告系列 |
| 每广告系列 API token | $7–28 | 长广告生成 prompt 的输入 + 输出 token |
| 运营方时间 | 3–5 小时 | QA、转化追踪、上线 |
| 获客成本 | 每个客户 $30–150 | 冷触达、付费广告、推荐 |
| **总交付成本** | **$60–380** | 不含运营方小时费率 |

按 $500 交付价算，毛利 24–88%，取决于运营方效率和 CAC。按 $800 算，毛利 53–90%。要拿到毛利上限，运营方得跑每月 10+ 个广告系列，把 Claude 订阅和 QA 摊薄。

## 谁买这个

三类买家，价格敏感度不同：

1. **月广告支出 $10k–100k 的小电商商家。** 要比代理更快的迭代循环。价格承受 $300–800 每个广告系列，4–10 个广告系列的月 retainer。
2. **绩效营销顾问。** 每季度给自己客户交付 20–50 个广告系列，要压每个广告系列的时间。价格承受 $100–300 每个广告系列作为白标输入，他们再按完整代理价 ($1,000–3,000 每个) 转卖。
3. **SaaS 营销团队。** 内部跑付费搜索但想跳过首稿。价格承受 $200–500 每个广告系列，内部走预算。

三类买家对应三种不同的 go-to-market 动作：直客内容营销、代理合作、SaaS 社群分销。没有一种是自动的。

## 套壳生意坐在哪里

三种可行形状：

1. **直接交付。** 每个广告系列 $300–800，按项目计或按月 retainer 4–10 个广告系列。最贴合小电商买家。
2. **白标交付。** 同一套搭建卖给代理 $100–300 每个，代理再按完整代理价转卖客户。最贴合顾问买家；代理吃掉毛利，运营方吃掉量。
3. **模板转售。** 精挑细选的广告系列模板库 $50–200/月每个订阅。最贴合 SaaS 营销团队买家；买家从运营方已经 QA 过的模板自助。

三种形状可以从同一个 Claude Code 搭建平行跑。搭建是一样的；go-to-market 和定价不同。

## 案例漏掉的

帖子展示搭建几分钟跑完。决定一个广告系列实际表现得不得分的工作，帖子里都没有：

1. **QA 循环。** 每条生成的标题都要人工过一遍商标、政策、品牌语气适配。标准拒收率 10–30%。帖子没点这个。
2. **转化追踪接线。** 一个没有配好 GA4 转化事件的 Google Ads 广告系列不会正确优化。接线不在搭建里，每个广告系列 1–3 小时运营方工作。
3. **上线后调优。** 关键词裁剪、出价调整、搜索词报告审核是每个活跃广告系列每周 4–8 小时的工作。广告系列实际是赢是输，就在这里。
4. **Google Ads policy 审核。** 生成的标题经常撞商标或「不切实际承诺」被拒，要人工重写。标准重写率 5–15%。

搭建把首稿工作压完了。首稿之后的工作决定广告表现得不得分。帖子卖的是搭建。生意是上线后调优。

## 带走就一句

Claude Code 广告搭建器是真产品，周围有三个可行的套壳生意。搭建把 6–10 小时的代理工作压到几分钟；每个广告系列剩下的 3–5 小时运营方工作才是真正的产品。$500+ 交付价下套壳毛利很健康；对手是 DIY 买家的护城河是 QA + 转化追踪 + 上线后调优循环，不是搭建本身。

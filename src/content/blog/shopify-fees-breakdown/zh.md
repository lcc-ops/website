Shopify 头条数字是 "$39/月"。真实成本远高于此。下面是 2026 年一家典型店铺的每一笔费用。

## 订阅档

| 套餐 | 月费 | 适合 |
|---|---|---|
| Starter | $5 | 副业、线下销售 |
| Basic | $39 | 新店，月营收 < $10k |
| Shopify | $105 | 增长期，需要更多报表 |
| Advanced | $399 | 年营收 $1M+，最低刷卡费率 |

三年计划 Basic 降至 $29/月，Advanced 降至 $299/月。

## 交易费 — Shopify Payments

用 Shopify Payments（底层 Stripe）：

| 渠道 | 在线 | 线下 |
|---|---|---|
| Basic 套餐 | 2.9% + $0.30 | 2.7% + $0.00 |
| Shopify 套餐 | 2.7% + $0.30 | 2.5% + $0.00 |
| Advanced 套餐 | 2.5% + $0.30 | 2.4% + $0.00 |

国际卡（非美国）在百分比上额外加 **1.5%**。

## 交易费 — 第三方网关

用 PayPal、Stripe 直连或其他网关：

- Basic 套餐：每笔额外 **2.0%**
- Shopify 套餐：1.0%
- Advanced 套餐：0.5%

算账：Basic + PayPal = 3.49% + $0.30 + PayPal 费率。$10k/月以下就用 Shopify Payments，别折腾。

## $50 一单实际到手

假设 Shopify Basic + Shopify Payments，无应用额外抽成：

```
毛收入：                 $50.00
Shopify Payments 费率：  −$1.75  （2.9% + $0.30 = $1.45 + $0.30）
订阅分摊：               −$0.65  （$39 ÷ 60 单/月）
                          ──────
COGS 之前到手：          $47.60
```

$50 一单平台成本 4.8%。$20 一单 7.4%。$200 一单 3.6%。

## 多数卖家漏掉的隐藏费

### 1. 货币转换

Shopify Payments 对非美元交易额外收 **1.5%**，再加 1–2% 的汇兑点差。尽量用美元结算。

### 2. 退款费

Shopify 交易费在退款时**不退**。服装类影响大 — 15% 退货率意味着你付了 $1.75 交易费处理订单，退款后这 $1.75 白付。

### 3. 拒付费

每笔拒付 **$15**，不管输赢。哪怕赢了，你也付了 $15 自卫。

### 4. 应用交易费

部分应用（订阅、批发、市场）在 Shopify Payments 之上额外收 **2%**。检查应用栈。

### 5. 主题费

免费主题功能有限。多数正经店铺一次性付 $150–350 买付费主题。避开有隐藏授权的"免费主题"。

### 6. 邮件成本

Shopify Email 前 10,000 封/月免费，超出 $0.001/封。Klaviyo 集成 500 联系人后起步 $45/月。

## 真实店铺的月度账单

| 项目 | 成本 |
|---|---|
| Basic 订阅 | $39 |
| 付费主题（摊销） | $12 |
| 5 个必备应用（平均） | $90 |
| 邮件工具（1k 联系人） | $30 |
| 域名 | $1 |
| **固定合计** | **$172/月** |
| 交易费（$10k 营收） | ~$320 |
| **平台总成本** | **$492** |

占总营收 **4.9%**。定价时直接算进去。

## 什么时候升级套餐

Basic 继续用，直到：

- 月流水 $10k → Shopify 套餐用更低的交易费率回本
- 需要第三方计算运费
- 需要礼品卡

年营收 $1M+ 时升 Advanced，$399/月被更低的费率和更好报表覆盖。

## 前 $5k/月免费替代

- **Stripe + 免费落地页** — 总费用不到 $50/月
- **Gumroad** — 10% 一次性，无月费
- **Payhip** — 5% + $0 固定，无月费

验证期用这些。需要库存、物流标签或员工账号时再换 Shopify。

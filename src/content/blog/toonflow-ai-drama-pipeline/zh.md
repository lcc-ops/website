一条推文拆的是 Toonflow——一个 AI 短剧流水线,接一句一句话输入,出一部完整短剧(剧本、分镜、配音、剪辑全自动)。流水线跑 6 个 AI agent(导演、编剧、分镜、配音、质检、剪辑)像真实剧组一样分工。用户不用配环境、不用写 prompt;界面是拖拽分镜、调台词、换 BGM。头条技术声称:跨镜头、跨场景角色一致性——AI 视频最难的问题。案例不是「AI 视频有效」。案例是:2026 年 6 agent AI 短剧制作的单位经济,以及「角色一致性」突破究竟对海外短剧市场解锁了什么。下面是单集数学、角色一致性突破、以及案例没展开的四点。

## 6 agent 流水线

```
输入: 一句话 brief
   ↓
导演 agent:        解读 brief,定视觉风格,定角色规范
   ↓
编剧 agent:        起草剧本、台词、场景拆解
   ↓
分镜 agent:        生成逐镜头画面
   ↓
配音 agent:        生成台词、BGM、音效
   ↓
质检 agent:        查角色一致性、音频同步、场景逻辑
   ↓
剪辑 agent:        拼最终版、调色、转场
   ↓
输出: 完整短剧
```

导演 agent 把角色规范传给所有下游 agent。质检 agent 验证一致性。用户通过拖拽在分镜阶段介入,其余自动化。流水线用模型推理成本替代 5–10 人剧组。

## 角色一致性突破

AI 视频自 2023 年以来最难的问题是角色一致性。之前的模型每个镜头都把角色画得稍微不一样——脸、衣服、身材都不同——打破沉浸感。6 agent 流水线声称解决:

```
导演 agent:
  - 生成规范角色规范(脸、身材、衣服、声音)
  - 把规范存到流水线全局记忆
   ↓
分镜 agent:
  - 每个镜头读取规范
  - 以规范为约束重生成画面
   ↓
质检 agent:
  - 按规范验证每个镜头
  - 标记不一致让重生成
   ↓
剪辑 agent:
  - 拼接镜头;拒绝任何没过质检的
   ↓
输出: 过质检的镜头 → 角色在所有镜头里都长一样
```

突破不在模型——在编排。导演 agent 的规范 + 质检 agent 的验证,是给跨镜头一致性的东西。案例是公开声称 6 agent 流水线跨镜头一致性的首例。独立基准还没验证。

## 单位经济

```
单集 AI 成本:                  5–50 美金(模型 + 流水线)
单集分发成本:                  0–50 美金(CAC 到付费观众)
单集营收(海外):                1K–10K 美金(5–10 倍中国平台)
单集毛利:                      90–99%
70 集单剧毛利:                 70K–700K 美金
70 集 AI 成本:                 350–3500 美金
70 集分发成本:                 0–3500 美金
```

案例没给单集成本。隐含 5–50 美金。AI 成本压缩相对传统制作(单 70 集 5K–50K)10–100 倍。瓶颈不是成本;瓶颈是分发。

## 为什么角色一致性是解锁

角色一致性问题在 2026 年是 AI 短剧采用最大的单一阻塞。operator 能便宜出 70 集,但镜头之间主角脸变,观众不容忍,平台不推。

如果 Toonflow 的角色一致性声明成立,解锁是体量游戏。operator 一周能出 1–3 部,接近零生产成本,知道每部都沉浸稳定。72 小时海外营收曲线按周复利。复利是解锁。

如果声明不成立,Toonflow 是个产出 70 集沉浸破碎内容的工具。买家(operator)在质检里发现这事,因为制作成本已沉,还是出货,看 72 小时曲线表现不及预期。质检环节是隐性风险。

## 案例没说的事

四点。

1. **单集成本。** 案例没给美元数。隐含 5–50 美金。
2. **失败模式。** 6 agent 流水线产出不一致镜头时怎么办;用户手动覆盖有限。案例没处理质检 agent 标记一个镜头时用户做什么。
3. **质量上限。** 案例框定 6 agent 编排是突破,但没处理输出质量的上限(电影级短剧 vs 竖屏短剧)。6 agent 流水线针对竖屏格式优化;电影级是另一回事。
4. **分发 CAC。** 案例没处理拉到付费观众的成本;生产成本是单位经济的小头。案例忽略分发瓶颈。

## 结论

案例不是「Toonflow 能做短剧」。案例是:一个解决角色一致性的 6 agent AI 视频流水线,是 AI 短剧体量游戏的解锁。operator 一周能出 1–3 部,接近零生产成本,知道输出沉浸稳定。72 小时海外营收曲线按周复利。复利是解锁——如果角色一致性声明在生产中成立。

看这条案例的多数人,带走就一句:如果你是 AI 短剧 operator,生产成本不再是瓶颈。分发 CAC 是。把时间花在分发上,不是生产上。如果 Toonflow(或同类 6 agent 流水线)给你跨镜头一致性,一周出 1–3 部,让 72 小时曲线复利。

多数卖家凭感觉定价：*成本 × 3 差不多*。直到第三个月账目显示亏损。下面是专业跨境卖家真正用的公式。

## 公式

```
售价 = (成本 + 物流 + 固定费) ÷ (1 − 佣金 − 支付费率 − 目标利润率)
```

其中：

- **成本** = 美元采购价
- **物流** = 单笔到客户成本
- **固定费** = 单笔平台费（Shopify $0.30，PayPal 部分市场 $0.49）
- **佣金 + 支付费率** = 合计百分比费用
- **目标利润率** = 想留的净利率 %

## 算例 1 — Shopify 品牌商品

- 成本：$8
- 物流：$3.50
- Shopify Payments 费率：2.9% + $0.30
- 目标利润率：35%

```
售价 = (8 + 3.5 + 0.3) ÷ (1 − 0.029 − 0.35)
售价 = 11.8 ÷ 0.621
售价 = $19.00
```

单件利润：$19.00 × (1 − 0.029) − 11.80 = **$6.65**

## 算例 2 — Etsy 手工艺品

- 成本：$12（材料 + 工时）
- 物流：$5（含 $1 包装）
- Etsy 交易费：6.5%
- Etsy 支付处理：3% + $0.25
- 目标利润率：40%

```
售价 = (12 + 5 + 0.25) ÷ (1 − 0.065 − 0.03 − 0.40)
售价 = 17.25 ÷ 0.505
售价 = $34.16
```

## 算例 3 — 亚马逊 FBA

- 成本：$6
- 亚马逊推荐费：15%
- FBA 履约：$4.50
- 目标利润率：25%

```
售价 = (6 + 4.5) ÷ (1 − 0.15 − 0.25)
售价 = 10.5 ÷ 0.60
售价 = $17.50
```

盈亏平衡价（0% 利润）是 $13.13。超过它卖就是赚，低于它卖就是亏。

## 为什么加价倍数会失灵

$8 成本 × 3 = $24。但实际：

| 项目 | 金额 |
|---|---|
| 收入 | $24.00 |
| Shopify 费率（2.9% + $0.30） | −$1.00 |
| 支付处理（已含上面） | — |
| 物流 | −$3.50 |
| 成本 | −$8.00 |
| **净利润** | **$11.50** |

净利率 48%，不是你想要的 35%。3× 倍数会让低价品溢价过度，高价品反而不够。公式在全目录都能正确缩放。

## 考虑退货调整

按品类在成本上加 5–15% 退货缓冲：

| 品类 | 退货率缓冲 |
|---|---|
| 服装 | 15% |
| 电子 | 8% |
| 家居 | 6% |
| 美妆 | 4% |
| 图书/音像 | 3% |

算例 1 加 15% 退货，成本从 $8 调到 $9.20：

```
售价 = (9.20 + 3.5 + 0.3) ÷ (1 − 0.029 − 0.35)
售价 = $19.95
```

## 考虑广告调整

跑付费流量时，实际目标利润率更低。35% 利润率配 20% 广告成本占比，实际净利只有 15%。要么提价，要么降 ACoS。

Etsy 和 Shopify 看起来干同一件事——卖手作——但费用结构每个条目都不一样。下面是 $35 产品的真实成本，以及双方各占上风的销量临界点。

## $35 产品费用堆栈

| 费用 | Etsy | Shopify Basic + Stripe |
|---|---|---|
| 月费 | $0 | $39/月 |
| 上架 / 商品页 | $0.20 | $0 |
| 成交费 | 6.5% = $2.28 | 0% |
| 支付处理 | 3% + $0.25 = $1.30 | 2.9% + $0.30 = $1.32 |
| **平台 + 支付合计** | **$3.78** | **$1.32** |
| 站外广告（归因单 15%） | $5.25（被归因时） | $0 |
| **含站外广告合计** | **$9.03** | **$1.32** |
| 运费标签折扣 | USPS 9 折–7 折 | 视情况 |

低销量 Etsy 单价更低，但站外广告和 $0.20 上架会把算账翻过来。

## 上架费细节

每个 Etsy 上架 $0.20。4 个月或售出时自动续。

- 100 个在线 listing，4 个月没卖出：续期费 $20
- 100 个在线 listing，4 个月全卖出：$0（卖出即续）

SKU 多、动销慢的卖家，光上架就吃钱。1,000 SKU 的店每 4 个月花 $200 才能保持可见。

## 成交 + 支付费算账

| 订单金额 | Etsy（6.5% + 3% + $0.25） | Shopify Basic + Stripe（2.9% + $0.30） |
|---|---|---|
| $10 | $1.43 | $0.59 |
| $20 | $2.23 | $0.88 |
| $35 | $3.78 | $1.32 |
| $50 | $5.00 | $1.75 |
| $100 | $8.75 | $3.20 |

每个价位 Etsy 抽的"平台 + 支付"都差不多是 Shopify 的 3 倍。

## 站外广告

Etsy 在平台外投广告（Google、Facebook、Pinterest）。年营收 $10K 以下，被归因的订单收 15%。$10K 以上降到 12%。

$10K 以下不能退出。这个区间里站外广告能把 Etsy 从"便宜"变成"贵很多"。

$35 产品 30% 归因率，每归因单加 $5.25。月 100 单、30% 归因 = $157.50/月，已经是 Shopify 月费的 4 倍。

## 运费标签折扣

Etsy 提供 USPS、FedEx、Canada Post 标签，9 折–7 折。$5 USPS Priority 标签，平台付 $3.50–$4.50。

Shopify Shipping 提供 USPS、UPS、DHL 等同价位。标签成本两边差不多，差别在平台费上加多少。

## 盈亏平衡：Etsy vs Shopify

$35 产品、不算站外广告：

```
Etsy 成本 = 0.20 + 0.065 × 35 + 0.03 × 35 + 0.25 = $3.78
Shopify 单订单成本 = 39 / n + 0.029 × 35 + 0.30 = 39/n + 1.32
```

平衡点：3.78 = 39/n + 1.32，n = 39 / 2.46 ≈ 16 单/月。

月单 16 以下 Etsy 纯费用更便宜。超过 Shopify 占优。

加上站外广告后：

```
Etsy 含广告 = 3.78 + 0.15 × 35 = $9.03
```

平衡点：9.03 = 39/n + 1.32，n = 5 单/月。月单 5 以上 Shopify 更划算。

## 怎么选

**Etsy 占优：**

- 月单 5–10 以下
- 产品靠发现（古董、垂直手作）
- 新启动、想用免费基建
- 客单价 $20 以下（站外广告对低价不敏感）

**Shopify 占优：**

- 月单 10 以上
- 需要邮件列表、自定义域名、弃单挽回
- 自己投广告
- 想做一个换平台也活着的品牌

**两边都开：** 很多卖家 Shopify 跑主站，Etsy 跑曝光。在 Etsy 买，复购转独立站。代价：1.5× 费用；收益：自有客户列表。

## 两边的隐藏成本

| | Etsy | Shopify |
|---|---|---|
| 退货 | 大部分品类卖家付退货运费 | 卖家付，除非用退货 app |
| 货币转换 | 非美元销售 1.5% | 2%（Shopify Payments 多币种） |
| 拒付费 | 每笔 $15 | 每笔 $15 |
| 工具 / 应用栈 | $0–$30/月（Etsy 工具） | $50–$200/月（典型栈） |

Shopify 的应用栈是隐藏的坑。大多数店还没意识到就月花 $100–$200。

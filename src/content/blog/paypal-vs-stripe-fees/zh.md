PayPal 和 Stripe 表面看起来差不多。真实成本差距在国际销售和货币转换上才显现。

## 标准美国卡费率（2026）

| 项目 | PayPal Checkout | Stripe |
|---|---|---|
| 在线交易 | 3.49% + $0.49 | 2.9% + $0.30 |
| 线下（刷卡） | 2.29% + $0.09 | 2.7% + $0.05 |
| 月费 | $0 | $0 |
| 开通费 | $0 | $0 |
| 拒付费 | $20 | $15 |

PayPal 标准在线费率是 **3.49% + $0.49**（常见说法 "2.99% + $0.49" 是旧费率）。Stripe 年流水 $1M 以下是 **2.9% + $0.30**。

$50 一单差距：
- PayPal：$2.24
- Stripe：$1.75

Stripe 每单多留 $0.49。

## 小额费陷阱

两家都收单笔固定费。低价品上这就是巨大的百分比：

| 售价 | Stripe 成本 | 占销售 % |
|---|---|---|
| $5 | $0.45 | 8.9% |
| $10 | $0.59 | 5.9% |
| $25 | $1.03 | 4.1% |
| $50 | $1.75 | 3.5% |
| $100 | $3.20 | 3.2% |

$10 以下，支付成本就超过营收 6%。小额品要么捆绑销售，要么换 Square 这种扁平费率（线上 2.9% + $0.30，线下 2.6% + $0.10）。

## 国际卡费率

| | PayPal | Stripe |
|---|---|---|
| 国际卡附加 | 1.50% | 1.0%（在标准费率之上） |
| 货币转换费 | 3–4%（零售汇率） | 1%（批发汇率） |
| 支持货币 | 25 | 135+ |
| 提现货币 | 56 | 50+ |

$100 单卖给英国客户用英镑付：
- PayPal：3.49% + $0.49 + 1.5% + 3.5% 汇兑 = **$8.49 总计**
- Stripe：2.9% + $0.30 + 1% + 1% 汇兑 = **$5.70 总计**

Stripe 这一单省 **$2.79**。

## 拒付处理

| | PayPal | Stripe |
|---|---|---|
| 拒付费 | $20 | $15 |
| 网络费（胜诉） | — | $0.40（美国）/ $15（其他） |
| 申诉窗口 | 10 天 | 7–14 天 |
| 证据上传 | 手动表单 | 后台 + API |
| 胜诉率（典型） | ~40% | ~40% |

Stripe 后台和 API 让取证更容易。$0.40 网络费仅限 Visa/Mastercard（不含 Amex）。Stripe $15 拒付费胜诉后退；PayPal 的 $20 无论输赢都不退。

## 提现时效

| | PayPal | Stripe |
|---|---|---|
| 标准提现 | 1–3 工作日 | 2 工作日（Standard） |
| 实时提现 | $1.50/笔 | 1% 提现额（最低 $0.50） |
| 新账户冻结期 | 最长 21 天 | 7–14 天 |
| 冻结触发 | 高拒付率、突增交易量 | 同上 |

两家都冻结新账户。Stripe 冻结条件更透明。

## 买家保护

| | PayPal 买家保护 | Stripe（无买家保护） |
|---|---|---|
| 未授权购买 | 覆盖 | 不覆盖（用 3DS） |
| 未收到货 | 覆盖 | 不覆盖 |
| 货严重不符 | 覆盖 | 不覆盖 |
| 对卖家影响 | 拒付更高 | 争议率更低 |

PayPal 买家保护能拉高转化，但也拉高拒付（特别是"货不符"投诉）。Stripe 靠 3D Secure 反欺诈，把责任转移给发卡行。

## 隐藏成本

### PayPal

- **货币转换点差**：中端汇率之上 3–4%
- **跨境费**：标准费率之上 1.5%
- **不活跃费**：商业账户无
- **退款费**：不退（原始交易费照付）
- **小额支付费率**：4.99% + $0.09（$5 以下销售更划算）

### Stripe

- **国际卡费**：标准费率之上 1.0%
- **货币转换**：1%（接近中端汇率）
- **ACH 直接借记**：0.8%，上限 $5
- **SEPA 直接借记**：€0.35/笔
- **订阅计费**：定期收费 +0.5%（开票免费）
- **Stripe Tax**：交易额 0.5%

## 怎么选

**用 Stripe 如果：**
- 月流水 > $10k（$80k/月时费率更优）
- 卖给国际非美元客户
- 想要对开发者友好的 API
- 做订阅业务

**用 PayPal 如果：**
- 客户期望 PayPal（尤其欧盟和亚洲）
- 在需要 PayPal 的平台上卖
- 卖 $5 以下商品（小额费率更优）
- 想用买家保护拉转化

## 实操配置

多数跨境卖家**两个都用**：Stripe 为主，PayPal 备用。Shopify Payments（底层 Stripe）给你 Stripe 的费率；额外加 PayPal 作为备选网关。客户在结账时选。

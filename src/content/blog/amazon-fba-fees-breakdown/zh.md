Amazon 的费率表 14 页，多数卖家定价只算了推荐费，剩下的藏在履约、仓储、退货、长期库存附加费里。下面是一款 $25 产品的完整账单。

## $25 产品的完整费用堆栈

| 费用 | 金额 | 占售价比 |
|---|---|---|
| **售价** | **$25.00** | **100%** |
| 推荐费（15%） | $3.75 | 15% |
| FBA 履约费（小标件，≤6 oz） | $3.06 | 12.2% |
| FBA 履约费（小标件，6–12 oz） | $3.43 | 13.7% |
| FBA 履约费（小标件，12–16 oz） | $3.86 | 15.4% |
| 关闭费（仅限图书、媒体、电子游戏） | $0.00 | 0% |
| **Amazon 的总抽成（不含仓储）** | **$7.61** | **30.4%** |

这是底线。下面是大多数人会忘记的附加费。

## 仓储费

Amazon 按每立方英尺每日收取库存占用费。2026 年标准：非旺季（1–9 月）$0.87/立方英尺/月，旺季（10–12 月）$2.40/立方英尺/月。

以 12×8×4 英寸的盒子（约 0.222 立方英尺）装 $25 产品：

- 非旺季月仓储：$0.19
- 旺季月仓储：$0.53

仓储费单看小，但会累积。1,000 件的产品花 6 个月才卖完，每月每件扣 $0.10，6 个月就是 $600。

## 长期库存附加费

FBA 仓库存放超过 181 天的单位每月加收附加费：

| 存放天数 | 小标件附加费 |
|---|---|
| 0–180 | $0 |
| 181–270 | $0.30/件/月 |
| 271–330 | $0.60/件/月 |
| 331–365 | $0.90/件/月 |
| 365+ | 长期仓储费：$6.90/立方英尺（或 $0.15/件，取高） |

一款 9 个月才清掉的货，光长期库存附加费就 $1.80——还没算仓储。

## 退货处理费

退货 Amazon 照收推荐费，并对特定品类（服饰、鞋、手表、珠宝、箱包及大部分消费品）加收退货处理费。

退货处理费取以下较低者：

- 原始 FBA 履约费的 100%
- $5.00

对 $25 产品（$3.86 履约费），每退一件 Amazon 不亏反赚 $3.86 加推荐费。退货率 10% 的产品，每卖一件平均被退货吃 $0.77。

## 移除与弃置

要清掉死库存的话：

- 移除订单：$0.75/件（标准）或 $1.50/件（超大件）
- 弃置订单：$0.30/件（标准）或 $0.60/件（超大件）

500 件死库存，移除就要 $375。有些卖家把这一项当作"赌错 SKU 的成本"。

## 算账：30% 净利率反推售价

目标：覆盖所有 Amazon 费用和货成本后，净利 30%。

以 $25 小标件、8% 退货率、6 个月仓储、$6 到岸成本为例：

```
净利率 = 0.30
推荐费 = 0.15
履约费 = 3.86
退货损耗 = 0.08 × (3.86 + 3.75) = 0.61
仓储 = 0.40
附加费 = 0.50

售价 = (成本 + 履约 + 仓储 + 附加 + 退货损耗) ÷ (1 − 推荐费 − 净利率)
售价 = (6 + 3.86 + 0.40 + 0.50 + 0.61) ÷ (1 − 0.15 − 0.30)
售价 = 11.37 ÷ 0.55
售价 = $20.67
```

$20.67 时，所有费用 + 8% 退货率都能消化，净利 30%。低于这个售价，在退货多的月份就是亏。

## 最能撬动利润的几个杠杆

| 杠杆 | $25 产品影响 |
|---|---|
| 退货率从 5% 升到 10% | −$0.38/件 |
| 长期库存（181+ 天） | −$0.30/件/月 |
| 改 FBM 自发货 | 省 $3.86，转化率掉约 20% |
| 多渠道履约（MCF） | 加 $0.50–$1.50/件 |
| 加 $5 优惠券 | 推荐费省 $0.75，但毛利少 $4.25 |

## FBA vs FBM — 什么时候切换

FBA 占优的情况：

- 产品小、轻、对转化敏感
- 想要 Prime 曝光
- 退货率低于 8%

FBM 占优的情况：

- 超大件、滞销、退货率超过 12%
- 自发货成本能压到 $3.86/件以下
- 仓储附加费正在吃利润

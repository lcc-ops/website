一条爆款 X 帖（211 赞、15 回复、43 转推）介绍了 CashClaw，一个开源 AI Agent，自己找活、自己报价、自己执行、自己交付、自己开票。搭建是真的、开源的、一个周末能 fork。套壳生意是问题。

## 搭建做了什么

| 阶段 | 在做什么 | 谁决策 |
|---|---|---|
| 找任务 | 扫招聘板、平台列表或直接客户请求 | Agent |
| 报价 | 估算小时数、报价 | Agent（可选运营方覆盖） |
| 执行 | 调 LLM 产出交付物 | Agent |
| 交付 + 开票 | 发交付物，开票收钱 | Agent |

运营方把钱包和目标接好之后，循环不需要人介入。运营方的角色坍缩成：(a) 设定目标；(b) 监控循环；(c) 处理冒出来的争议。

## 每个任务的经济学

| 项 | 成本 | 备注 |
|---|---|---|
| Claude 订阅 | $20–200/月固定 | 分摊到每月 100–1,000 个任务 |
| 每个任务 LLM API | $0.05–2 短，$5–20 长 | 看交付物长度 |
| 算力 + 存储 | $5–30/月每个 Agent | 小 VPS，SQLite |
| 平台费 | 发票金额的 5–20% | 每个平台不同 |
| **总交付成本** | **每个任务 $0.10–25** | 加运营方监控时间 |

每个任务开票金额 $20–500 看平台。中位数 $50，毛利 50–99%（不含拒付 / 撤单损失）。经济学只在量上成立——一天一个任务摊不掉 Claude 订阅。

## 买家画像

三群人，价格敏感度和风险承受不同：

1. **单人 founder。** 要一个 24/7 不用人盯的获客或内容生产循环。价格承受 $200–500/月一个托管 Agent，每天回个报告。
2. **小代理。** 要用 Agent 替代兼职外包，处理高量低风险任务（数据录入、短文案、基础调研）。价格承受 $500–2,000/月每个 Agent 实例。
3. **副业运营方。** 要 Agent 每天测 50+ 个小单，留下付钱的。价格承受 $50–200/月，或者分成。

这三类都需要同一件帖子没点的事：Agent 发票背后的法人实体。

## 法人实体的缺口

大多数平台（Upwork、Fiverr、Toptal、直接客户开票平台）要求一个注册的商业实体——LLC、个体工商户或同等——才能收款。一个自开票 AI Agent 在大多数司法管辖区不能以个人身份开票，背后必须有一个税务登记的实体。

帖子没点这个要求。CashClaw 周围的套壳生意坐在要么：(a) 运营方自己的 LLC；要么 (b) 一个代 Agent 处理开票、税务、争议的壳公司。壳公司形状更耐用，因为它把拒付 / 撤单风险按量吸收掉。

## 三个失败模式

1. **Agent 在难任务上报价低于成本。** 没有报价硬底线，Agent 会在复杂任务上卷到底价，执行质量差，在发票上亏钱。缓解措施是按平台设报价下限，加一个交付前的质量检查。
2. **Agent 被平台限流或封号。** 大多数平台封每天发 10–20+ 报价、或交付质量低于阈值的账号。缓解措施是按平台限速，加一个低置信度交付的拦截闸。
3. **Agent 交付错误或商标违规的产物，触发拒付。** Agent 生成交付物的标准拒付率 5–15%。缓解措施是任何高于可配置金额阈值的交付都加人工 QA。

## 带走就一句

CashClaw 是真开源搭建，有清晰的套壳生意路径。经济学在量上成立：每月 100+ 任务、中位数 $50，能月入 $5,000+ 毛利（不含拒付损失）。对手是 DIY 买家的护城河是法人实体、争议处理、按平台调 prompt——不是 Agent 本身。帖子卖的是循环。生意是让循环能开票的壳。

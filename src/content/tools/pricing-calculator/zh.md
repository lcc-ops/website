**产品定价计算器** 回答每个跨境卖家都问的问题：*我应该卖多少钱才真赚钱？*

多数定价公式只算 "成本 × 加价倍数"。但这样漏掉了物流、平台佣金、支付手续费，还有那些悄悄吃掉利润的单笔固定费。这个计算器四个都包含。

## 使用方法

1. 填 **产品成本**（付给供应商的金额，美元）。
2. 填 **物流成本 / 单**（包邮模型填 0）。
3. 填 **佣金 %**（美国 Shopify Payments 是 2.9%；Etsy 是 6.5% + 3%；亚马逊是 15%）。
4. 填 **支付处理 %**（如果平台打包在一起，和佣金一样）。
5. 填 **单笔固定费**（Shopify 收 $0.30；PayPal 部分地区收 $0.49）。
6. 设 **目标净利率**（所有费用之后）。
7. 看 **建议售价**、**盈亏平衡价**、**单件利润**。

## 公式

```
售价 = (成本 + 物流 + 固定费) ÷ (1 − 佣金% − 支付费率% − 目标利润率%)
```

盈亏平衡价假设 0% 净利率，超过它卖出去的部分都是利润。

## 算例

$8 产品 + $3.50 物流 + 2.9% 佣金 + 2.9% 支付费 + $0.30 固定费 + 30% 目标利润率：

- 建议售价：**$24.85**
- 盈亏平衡价：**$13.05**（零利润）
- 单件利润（按建议售价）：**$7.46**

如果你的市场卖不到 $24.85，目标利润率就不现实。要么砍成本、要么提价、要么接受更薄的利润。

## 这个计算器没算的

- **退货成本**（普通品类在成本上加 5–15%）
- **广告成本占比**（跑付费流量时从利润里扣）
- **汇兑损耗**（收美元花人民币时 1–3%）

## 常见错误

| 错误 | 修法 |
|---|---|
| 用毛利率而不是净利率 | 本计算器用净利率（扣完所有费） |
| 漏算单笔固定费 | 一定要加，规模大了累计惊人 |
| 公式里忽略物流 | 物流是成本，不是营销条目 |
| 货币混着用 | 先全部换算成同一种货币 |
